## Dadas as Linguagens Regulares sobre o alfabeto {a,b}, construa as Expressões Regulares correspondentes:

1. (a + b)aaa(a + b)

2. (a + b)*bb

3. (a + b)*bb(a + b)*aaaa

4. (a + b)aaa + bb(a + b) + ε

5. ab(a + b)*ba
